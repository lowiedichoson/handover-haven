# PEMI Website

## What It Is

VEI Website is the legacy website of Vantage Equities, Inc. It's a blog-like website where investors or clients visit to read the latest news and updates. It is built using a legacy version of WordPress and is currently being maintained by ICTG in updating / adding new public reports or files.
